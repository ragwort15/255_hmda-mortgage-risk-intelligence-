## Project Overview
Building an explainable mortgage risk prediction system using the 2022 HMDA dataset
filtered to California conventional home purchase loans. We compare Logistic Regression,
Random Forest, and XGBoost, with SHAP for explainability and a fairness analysis
across demographic groups.

## Dataset
2022 HMDA Data — Consumer Financial Protection Bureau (CFPB)
https://ffiec.cfpb.gov/data-browser/data/2022/

## Folder Structure
- `/data` — raw and processed dataset files
- `/notebooks` — Jupyter notebooks for EDA, preprocessing, modeling
- `/models` — saved model artifacts
- `/reports` — check-in and final report documents
